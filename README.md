### 快递查询 webAPP单页（react）
#### 该项目用到了create-react-app作为脚手架构建，搭配Ant Design Mobile进行开发。因为工作中用到的是Ant Design 桌面版，所以想用下移动版本的开发下移动端体验下效果。项目中已经配置了react-router，可以根据代码，配置自己喜欢的路由方式。
